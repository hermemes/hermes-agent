How it works

1

Define your KOL pool

Add BSC wallet addresses of KOLs you want to track in a simple markdown file.

2

Agent scans autonomously

Hermemes fetches transactions from BscScan, enriches with DexScreener market data, and builds 25+ dimension profiles.

3

Get actionable signals

Copy/fade/skip decisions with confidence scores, safety checks, and multi-KOL convergence alerts.

Run it with AI Gateway

Plug into any OpenAI-compatible API via OpenRouter.

gateway.ts 
import { generateText } from 'ai';

const { text } = await generateText({
  model: "openai/gpt-5.4",
  prompt: 'Analyze KOL 0xABC...789 on BSC — full 25-dimension profile.',
});
